# MountainCar-V0

## Usage

```bash
usage: main.py [-h] [-j]

optional arguments:
  -h, --help      show this help message and exit
  -j, --joystick  On/Off joystick mode
```

## Evaluation

The last model will be evaluated

```bash
python3 eval.py
```

## Requirements

Firstly, u need to have `python3.8`

```bash
python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
```
